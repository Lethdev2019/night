Night

Combining the simplicity of python with the type concepts of Clang

Night is an interpreted programming language that combines the simplicity of Python, with the type concepts of the C family. The main goal of Night was to design a intuitive and easy to learn language.

On this website, you'll find everything you need to get up and running with Night. Below is each page in detail.

A major feature of Night that is targeted towards beginners is the ablility to code right here in this website without any downloads or external packages. You can check out the Online IDE page for more information. And just like any other language, Night allows you to build it from source, with thorough instructions on how to get started. You can check that out in the Source Build page. Currently however, only Windows is supported.

This website also features a complete tutorial to the language Night, perfect for beginners or anyone first time here. There's also a reference guide for a more indepth look at specific syntax. Currently, both the Tutorial page and Reference guide reflect Night v3.0.

And finally, for extra information about this language, you can view the GitHub page for a link to the GitHub repository, as well as more information on the specs of Night.
